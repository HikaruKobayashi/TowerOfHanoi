# 使い方

`git clone https://github.com/HikaruKobayashi/TowerOfHanoi.git`

→ ブラウザに`index.html`ファイルをドラッグ&ドロップする。

# 遊び方

1. 一度に移動させることができる円盤は1つだけです。
2. 小さな円盤の上に大きな円盤を積み上げることはできません。
3. 最短で一番左から全ての円盤を一番右に移動させる為には、何回移動させれば良いでしょう？
